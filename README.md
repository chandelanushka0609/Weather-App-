
# Weather App (Python + Tkinter)

This is a desktop-based Weather Application built using Python and Tkinter.
The app fetches real-time weather data using the OpenWeatherMap API.

1. Features
   1.1 Real-time weather by city name
   1.2 Temperature and feels-like temperature
   1.3 Weather condition display
   1.4 Simple and clean GUI
   1.5 API-based data fetching

2. Technologies Used
   2.1 Python
   2.2 Tkinter
   2.3 Requests library
   2.4 OpenWeatherMap API

3. How to Run
   3.1 Install Python 3
   3.2 Install required libraries using:
       pip install requests pillow
   3.3 Replace YOUR_API_KEY_HERE in weather_app.py with your OpenWeatherMap API key
   3.4 Run the file:
       python weather_app.py

4. Project Use Case
   This project is suitable for beginners learning Python GUI development
   and API integration.
